호스팅의 혁신을 구성하는 기능

WordPress 자동 구성

nginx + PHP-FPM 8.3 기반으로 WordPress 코어를 자동 설치하고 관리자 계정을 발급합니다.

MariaDB/MySQL

표준 MariaDB/MySQL을 기본 DB로 사용하고, 백업은 mysqldump로 유지합니다.

phpMyAdmin

pma.도메인 라우팅과 임시 토큰 링크로 DB 관리 접속 정보를 제공합니다.

도메인/DNS 자동화

Cloudflare Zone, DNS record, Worker route 구성을 한 흐름으로 연결합니다.

캐시 관리

Cloudflare 캐시, Redis Object Cache, WordPress 캐시를 운영 화면에서 관리하도록 확장 가능한 구조입니다.

파일 관리자

테마, 플러그인, 업로드 파일을 다루는 파일 관리자 화면으로 바로 이동할 수 있습니다.
