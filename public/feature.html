Feature Overview

Cloudways 감성의 WordPress 운영 도구

서버, 애플리케이션, 도메인, 데이터베이스를 하나의 흐름으로 관리합니다.

Server

Launch Server

GitHub 레포 생성, WordPress 다운로드, SQLite DB 생성, Worker 배포를 자동화합니다.

App

WordPress Admin

wp-admin 접속 정보와 기본 URL을 호스팅 정보에서 즉시 확인할 수 있습니다.

DB

SQLite + PHPLiteAdmin

MySQL 없이 .db/.sqlite 파일 기반으로 관리하고 PMA/PLA 관리 UI를 제공합니다.

Domain

멀티 사이트 도메인 연결

하나의 계정 안 여러 호스팅에 Cloudflare Zone을 각각 연결하는 구조를 제공합니다.

Ops

Monitoring-ready UI

CPU, RAM, 트래픽, 상태 카드가 확장 가능한 SaaS 패널 구조로 준비되어 있습니다.

Security

Credential Vault

GitHub/Cloudflare 자격증명을 사용자별로 암호화 저장하고 배포 시에만 사용합니다.
