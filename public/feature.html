호스팅의 혁신을 구성하는 기능

WordPress 자동 구성

nginx + PHP-FPM 8.3 기반으로 WordPress 코어를 자동 설치하고 관리자 계정을 발급합니다.

MariaDB/MySQL

표준 MariaDB/MySQL을 기본 DB로 사용하고, 백업은 mysqldump로 유지합니다.

phpMyAdmin

pma.도메인 라우팅과 임시 토큰 링크로 DB 관리 접속 정보를 제공합니다.

도메인/DNS 자동화

Cloudflare Zone, DNS record, Worker route 구성을 한 흐름으로 연결합니다.

캐시 관리

Cloudflare 캐시, Redis Object Cache, WordPress 캐시를 운영 화면에서 관리하도록 확장 가능한 구조입니다.

파일 관리자

테마, 플러그인, 업로드 파일을 다루는 파일 관리자 화면으로 바로 이동할 수 있습니다.

기능 — wpspot
wpspot 로그인

기능

wpspot 인스턴스 하나로 호스팅, 캐시, 데이터베이스, 도메인까지 모두 관리해요.

서버리스 워드프레스 호스팅

GitHub Actions + nginx + PHP-FPM 8.3 + MariaDB/MySQL 조합으로 진짜 워드프레스 코어를 그대로 실행해요. VPS나 별도 서버 결제가 필요 없어요.

다중 도메인 연결

보유 도메인을 Primary로 연결하고, 추가 도메인은 Alias로 연결해 Primary로 301 리다이렉트할 수 있어요.

Cloudflare 엣지 캐시

전 세계 CDN 엣지에 정적 콘텐츠를 캐시해 빠르게 서빙하고, 인스턴스 관리 화면에서 버튼 한 번으로 즉시 초기화할 수 있어요.

phpMyAdmin 데이터베이스 관리

워드프레스 MariaDB/MySQL 데이터베이스를 웹에서 직접 조회·수정할 수 있어요. 접속은 항상 인증된 사용자만 가능하고, 토큰은 매일 자정(KST) 초기화돼요.

웹 파일 관리자

FTP 클라이언트 없이도 테마, 플러그인, 업로드 파일을 브라우저에서 바로 업로드·편집할 수 있어요.

상시 점검 & 자동 복구

GitHub Actions Keepalive가 60초마다 PHP 서버와 Cloudflare 터널 상태를 점검하고, 문제가 생기면 자동으로 재시작해요.

인스턴스 통합 대시보드

보유한 모든 인스턴스의 상태, CPU·RAM 사용량, 캐시 히트율, 평균 응답속도를 카드 하나로 모아볼 수 있어요.
