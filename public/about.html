About wpspot

복잡한 서버 관리를 쉬운 SaaS 패널로 바꾸는 프로젝트

wpspot은 cPanel처럼 복잡하지 않고 AWS 콘솔처럼 어렵지 않은 WordPress 호스팅 경험을 목표로 합니다.

운영 철학

  • Clean — 필요한 정보만 명확하게 표시합니다.
  • Professional — 실제 호스팅 운영에 맞는 상태/접속/도메인 정보를 제공합니다.
  • DevOps Friendly — GitHub Actions와 Cloudflare Workers를 기본 배포 레이어로 사용합니다.
  • Database Simple — WordPress 데이터베이스는 SQLite 파일 기반으로 유지합니다.
