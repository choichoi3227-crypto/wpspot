연동 정보를
입력해주세요

아래 3가지 정보가 있어야 사이트를 만들 수 있어요. 한 번만 등록하면 모든 사이트에 공통으로 사용돼요.

GitHub Token
repo, workflow 권한이 포함된 Fine-grained 또는 Classic PAT를 입력해주세요. 워드프레스 원본 레포 생성 및 GitHub Actions 실행에 사용돼요.
GCP (Blogger API) Token
Blogger API 스코프(https://www.googleapis.com/auth/blogger)가 포함된 토큰을 입력해주세요. 블로그스팟 템플릿 연동에 사용돼요.
Cloudflare Global API
블로그스팟 프록시 워커를 내 클라우드플레어 계정에 생성하는 데 사용돼요. Cloudflare 대시보드 > 프로필 > API 토큰에서 확인할 수 있어요.
