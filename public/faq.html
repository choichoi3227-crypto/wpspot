자주 묻는 질문

개인 도메인 없이도 생성할 수 있나요?

네. 기본 주소는 Cloudflare의 {worker}.{subdomain}.workers.dev 형태로 제공되고, 개인 도메인은 이후 연결합니다.

WordPress 데이터베이스는 무엇을 쓰나요?

MySQL이 아니라 SQLite 플러그인 기반의 wordpress.db 또는 호환 SQLite 파일을 사용합니다.

브라우저 WASM PHP를 사용하나요?

아니요. GitHub Actions의 PHP 8.3 내장 서버와 Cloudflare Workers 프록시 구조를 사용합니다.

하나의 계정에서 여러 사이트를 운영할 수 있나요?

네. instances 페이지에서 모든 호스팅을 카드 형태로 확인하고 각각의 instance-info 패널에서 관리할 수 있습니다.
