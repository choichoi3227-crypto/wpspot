호스팅의 혁신

워드프레스를 가장 빠르게 시작하는 방법

클라우드프레스는 GitHub Actions, Cloudflare Worker, nginx/PHP-FPM, MariaDB/MySQL, Redis를 하나로 묶어 WordPress 호스팅 생성부터 도메인 연결, phpMyAdmin 관리까지 자동화합니다.

워커 라우팅과 GitHub 워크플로우를 기반으로 반복 가능한 WordPress 운영 환경을 만듭니다.

깃허브와 클라우드플레어로 워드프레스 원본을 그대로 띄우고,
GitHub Actions, Cloudflare Worker, nginx/PHP-FPM, MariaDB/MySQL로 운영되는 WordPress 호스팅이에요.

엔터프라이즈급 성능을, 무료로

워드프레스 원본 파일과 폴더를 그대로 사용해요. 99.9%의 백엔드는 워드프레스가 담당하고, phpMyAdmin으로 MariaDB/MySQL 데이터베이스를 관리할 수 있어요.

필요한 건 토큰 3개뿐

GitHub Token, Cloudflare Global API Key, Cloudflare API Token을 내 계정에 등록하면 wpspot이 레포 생성부터 프록시 워커 배포까지 자동으로 처리해요.
